# Smart AgroBot

An IoT-based smart irrigation system using ESP32 and sensors for automated soil monitoring and watering.

## Features
- Soil moisture sensing
- Automated irrigation control
- Real-time data handling
- Basic robotics movement

## Tech Stack
- ESP32
- Sensors (moisture, environment)
- Firebase (for data handling)
